Математический анализ

f(x) = y = x * 2 – функциональная зависимость
При этом y называется функцией от x, а x - независимой переменной.
Запись f(x) призвана явно указать на функциональную зависимость.

Для каждого x должен быть определён максимум один y, или не определён вообще.

Область определения функции - это множество всех возможных значений x, для каждого из которых функция определена.

Виды простейших элементарных функций:
  1. Многочлены (полиномы):
    • По-другому называются целыми рациональными функциями.
    • В многочленах допускается только сложение, вычитание, умножение и возведение в целую положительную степень. Деление и извлечение корня отсутствует.
    • Если выражение состоит только из перечисленных выше арифметических операций над х и константами, то это выражение можно привести к многочлену.
    • Более сложные функции всегда стараются привести к многочлену.
  2. Рациональные функции:
    • Если к многочлену добавить деление, то получится рациональная (не целая) функция.
    • Любая рациональная функция может быть представлена в виде отношения двух многочленов:
      , где P(x) и Q(x) - многочлены, а Q(x) ≠ 0.
  3. Степенные функции:
    • y = xa, где a - константа.
    • Если а – целое, то это рациональная функция.
    • Если а – целое и a ≥ 0, то это многочлен (целая рациональная функция).
    • Если a – рациональная дробь , где p и q - целые числа, q > 0, то - иррациональная функция, поскольку к операциям, производимым над x добавляется извлечене корня степени q.
  4. Показательные функции:
    • y = ax, где a - положительная константа.
    • Показательные степени так называются, потому что переменная x является показателем степени, а константа a - основание степени.
    • Поскольку значение x не может быть вычислено посредством лишь алгебраических операций, то это не алгебраическая, то есть трансцендентная функция.
  5. Логарифмические функции:
    • y = logax, где a - константа-основание, a > 0 и a ≠ 1, х - показатель, x > 0.
    • Логарифм удовлетворяет выражению x = ay
    • Логарифм означает: в какую степень нужно возвести a, чтобы получить x?
    • Логарифм является обратной функцией для показательной функции.
    • Логарифм - трансцендентная функция.

  6. Тригонометрические функции:
    • y = sin/cos/tg/ctg/sec/cosec(x)
    • Эти функции являются периодическими.
  7. Обратные тригонометрические функции:
    • Это функции обратные к тригонометрическим.
    • Каждая тригонометрическая функция имеет бесконечное множество обратных функций (? благодаря своей периодиночти).
Ключевое правило БМВ: если какая-то величина x на протяжении всего процесса |x| < E, где E - любое положительное число, то эта величина - БМВ.
Результатом простейших алгебраических операций над бесконечно малой величиной будет бесконечно малая величина.
Сумма постояного числа БМВ есть величина БМ.
Если величина x на протяжении всего процесса равна нулю, то это БМВ, ведь ноль меньше любого положительного числа.

Если x - БМВ, то 1/x - ББВ.
Если x - ББВ, то 1/x - БМВ.

Если в каком-то процессе величина x бесконечно приближается к константе a, то расстояние между этими двумя величинами есть величина БМ, при этом a называется пределом x, и x стремится к a.
lim x = a или x → a – две равнозначные записи пределов.

Предел у величины может быть только один, при этом он должен быть константой в этом процессе.

В течение процесса, величина, приближаясь к своему пределу, может как угодно увеличиваться и/или убывать.

Величина, стремящаяся к пределу в каком-то процессе, ограничена в этом процессе.

Если в процессе x → a, a >/</≥/≤ 0, то начиная с какого-то момента x >/</≥/≤ 0.

Всякая БМВ имеет предел 0.

Всякая величина, пределом которой является 0, является БМВ.

БМВ – частные случаи величин, стремящихся к пределам.

У ББВ предела нет и не может быть.

Любая константа сама для себя вляется пределом.

Предел суммы постоянного числа величин равен сумме пределов.

Предел произведения постоянного числа величин равен произведению пределов.

Если x → a, k - постоянная велична, то kx → ka. lim(kx) = k lim(x)

Если x → a, n - любая постоянная натуральная величина, то xn → an.

>= ≥ != ≠

slope - коэффициент наклона прямой.
Он равен нулю, если прямая горизонтальна (параллельна x).
Он равен бесконечности, если прямая вертикальна (параллельна y).

+---------------+
| slope = y / x |
+---------------+-------------+
| x = cos точки на окружности |
| y = sin точки на окружности |
+-----------------------------+---------+
| tan(a) = tan(a + п) = tan(a + 180deg) |
+---------------------------------------+

sin - y - отношение противолежащего катета к гипотенузе
arcsin - производит опреацию, обратную sin: n = arcsin(sin(n))

cos - x - отношение прилежащего катета к гипотенузе
cos обратнопропорционален sin
arccos - производит опреацию, обратную cos: n = arccos(cos(n))

tan - sin/cos - отношение противолежащего катета к прилежащему - переводит радианы в slope (от 0 до бесконечности)
arctan (atan) - преобразование обратное tan: переводит slope в радианы

Math.atan(slope) - плохая по двум причинам: не отличает диаметрально противоположны углы и не может вычислить результат для углов 90 и 270 градусов
Math.atan2(y, x) - вариант лучше

Скалярная величина - величина, которая выражается одним значением.
Векторная величина содержит как минимум два значения.
Перемещение, скорость, ускорение - примеры векторных величин.

Скалярное произведение векторов (dot product):
A ∙ B = AxBx + AyBy   // результат скалярного произведения - число (скаляр)
A ∙ B = |A||B|cos(a)  // a - угол между A и B
Если dotProduct == 0, то A перпендикулярен B.
Порядок следования множителей не имеет значения.

Векторное произведение векторов (vector product):
A × B = [ AyBz - ByAz, AzBx - BzAx, AxBy - BxAy ]  // результат векторного произведения - вектор, перпендикулярный A и B
|A × B| = |A||B|sin(a)  // a - угол между A и B
При меремене мест в векторном произведении результирующий вектор изменит своё направление на проивоположное.

Нормаль - вектор, перпендикулярный данному.
В 2D: для нахождение нормали поменять местами x и y, затем поменять знак у одной из компонент.
В 3D: для нахождения нормали к двум перпендикулярным векторам, нужно найти их векторное произведение.
      применяется для нахождения нормали к плоскости.

-----------------------------------

Дифференцирование - нахождение производной функции для первообразной.
Интегрирование - нахождение первообразной функции для производной.

Неопределённый интеграл:
ИНТ f(x)dx = F(x) + C 
f(x) - подынтегральная функция
f(x)dx - подынтегральное выражение
F(x) - первообразная функция
x - переменная интегрирования
С - константа
F(x) + C - множество первообразных функций

Интеграл называется неопределённым, потому что имеет целую совокупность первообразных функций.
Решить неопределённый интеграл - это значит найти первообразную функцию.

Функция, имеющая первообразную в некотором промежутке, называется интегрируемой.
