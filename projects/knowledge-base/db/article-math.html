slope - коэффициент наклона прямой.
Он равен нулю, если прямая горизонтальна (параллельна x).
Он равен бесконечности, если прямая вертикальна (параллельна y).

+---------------+
| slope = y / x |
+---------------+-------------+
| x = cos точки на окружности |
| y = sin точки на окружности |
+-----------------------------+---------+
| tan(a) = tan(a + п) = tan(a + 180deg) |
+---------------------------------------+

sin - y - отношение противолежащего катета к гипотенузе
arcsin - производит опреацию, обратную sin: n = arcsin(sin(n))

cos - x - отношение прилежащего катета к гипотенузе
cos обратнопропорционален sin
arccos - производит опреацию, обратную cos: n = arccos(cos(n))

tan - sin/cos - отношение противолежащего катета к прилежащему - переводит радианы в slope (от 0 до бесконечности)
arctan (atan) - преобразование обратное tan: переводит slope в радианы

Math.atan(slope) - плохая по двум причинам: не отличает диаметрально противоположны углы и не может вычислить результат для углов 90 и 270 градусов
Math.atan2(y, x) - вариант лучше

декартова система -> полярная система: стр. 106

Скалярная величина - величина, которая выражается одним значением.
Векторная величина содержит как минимум два значения.
Перемещение, скорость, ускорение - примеры векторных величин.

Скалярное произведение векторов (dot product):
A ∙ B = AxBx + AyBy   // результат скалярного произведения - число (скаляр)
A ∙ B = |A||B|cos(a)  // a - угол между A и B
Если dotProduct == 0, то A перпендикулярен B.
Порядок следования множителей не имеет значения.

Векторное произведение векторов (vector product):
A × B = [ AyBz - ByAz, AzBx - BzAx, AxBy - BxAy ]  // результат векторного произведения - вектор, перпендикулярный A и B
|A × B| = |A||B|sin(a)  // a - угол между A и B
При меремене мест в векторном произведении результирующий вектор изменит своё направление на проивоположное.

Нормаль - вектор, перпендикулярный данному.
В 2D: для нахождение нормали поменять местами x и y, затем поменять знак у одной из компонент.
В 3D: для нахождения нормали к двум перпендикулярным векторам, нужно найти их векторное произведение.
      применяется для нахождения нормали к плоскости.
