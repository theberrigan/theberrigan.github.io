Высказывание — это какое-то утверждение, которое может быть истинно или ложно.
Простое высказывание — высказывание, состоящее из одного утверждения.
Сложное высказывание — высказывание, состоящее из нескольких утверждений.

Логические элементы

Импликация имеет следующие разновидности:

  • (p → q) — обычная импликация
  • (p → q) ∧ (q → p) — эквивалентность (↔)
  • q → p — конверсия
  • ~p → ~q — инверсия
  • ~q → ~p — контрапозиция

Теоремы часто удобно доказывать от противного, поэтому можно использовать контрапозицию, потому что она имеет такую же таблицу истинности, как и её импликация, чего нельзя сказать об инверсии и конверсии относительно их импликации.

Тавтология — высказывание истинное во всех случаях (p || !p). (он сдаст экзамен или не сдаст экзамен)
Противоречие — высказывание ложное во всех случаях (p && !p). (она идёт прямо и она не идёт прямо)
Отрицание тавтологии — противоречие.

Умозаключение состоит из ряда гипотез и одного заключения.
Правильное умозаключение — умозаключение, заключение которого истинно при истинности всех гипотез.
H1 && H2 && H3 → C (H1 — гипотизы, C — заключение)
Правило вывода — правильное умозаключение. Правила вывода используются для вывода новых теорем.


Битовая шкала - последовательность битов в каком-то типе.
Битовая шкала 2 ^ k - 1 содежит k бит, каждый из которых равен 1.

Инвариант обзначает:
- условие цикла, которое должно быть истинным для выполнения каждой итерации.
- условие, по которому формируется какая-то совокупность.

Алгоритм слияния подходит, когда элементы упорядочены.


МНОЖЕСТВА

Множество - неупорядоченный набор уникальных элементов.
Мощность множестве - его размер.
Мультимножество - множество, в которое один элемент входит несколько раз. Обычно связано со множеством, в которое эти же элементы входят по одному разу.

Предположим, дан универсум U = { 'a', 'b', 'c', 'd', 'e' }
Элементы U пронумерованы 1..n: 1 2 3 4 5
Тогда, битовая шкала C = 10011 обозначает подмножество A = { 'a', 'd', 'e' }
Каждый бит битовой шкалы обозначает наличие соответствующего по индексу элемента в подмножестве.
A ∓ B = битовая шкала, обозначающая пересечение множеств A и B.
A | B = битовая шкала, обозначающая объединение множеств A и B.
~A = битовая шкала, обозначающая дополнение множества A до U (U - A).
Если битовая шкала превосходит битность системы, то используется массив битовых шкал.

X = { 'a', 'b', 'c', 'd', 'e' } - множество элементов
Y = [ 2, 5, 1, 1, 100 ] - мультимножество над множеством X, каждый элемент которого обозначает количество вхождений соотвествующего элемента из X

Иногда требуется составить набор подмножеств множества А, которые удовлетворяют какому-то условию.
Проверка этого условия может быть слишком трудоёмким процессом. Чтобы сократить время проверки условия
для нового подмножества, можно использовать результат этой проверки для предыдущего подмножества.
Но для этого, предыдущее подмножество должно очень незначительно отличаться от текущего подмножества.
Поэтому, для последовательной генерации всех подмножеств множества, каждое их которых отличается от предыдущего
лишь одним элементом, можно использовать алгоритм Грея.
