Git

Перезапись всех локальных изменений данными с сервера: git fetch origin git reset --hard origin/master Обновить с сервера информацию о последних коммитах, ветках и изменениях: git fetch забирает последние изменения с сервера: git checkout Добавить румоут git remote add origin https://github.com/user/repo.git Чтобы изменить удалённый сервер git remote set-url origin git@github.com:user/repo.git !!! Не использовать HTTP, использовать SSH - https://github.com/WEMP/project-slideshow.git + git@github.com:WEMP/project-slideshow.git Просмотреть текущий ремоут git config --get remote.origin.url git remote show origin git remote -v Автопуш с датой и временем в сообщении коммита: git add -A git commit -m "$( date +"%d.%m.%Y %H:%M:%S" )" git push
